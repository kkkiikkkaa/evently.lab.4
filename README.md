Evently — Афіша подій

Evently — навчальний веб-проєкт, створений у рамках лабораторних робіт з дисципліни «Веб-орієнтована розробка ПЗ».
Сайт дозволяє переглядати події, зберігати їх у розділ “Мої події”, розгортати деталі та взаємодіяти з інтерфейсом через JavaScript.

 Функціональність

Перемикання світлої/темної теми (збереження у LocalStorage)

Простий клієнтський логін (збереження імені)

Додавання подій у розділ «Мої події»

Видалення подій з «Мої події»

Розгортання/приховування деталей подій

Динамічне додавання елементів і поточної дати

Акордеон «Показати більше»

Клієнтська валідація форми (ім’я, email, повідомлення)

Підсвітка навігації через JS

Зміна розміру шрифту клавішами (ArrowUp / ArrowDown)

 Технології

HTML5

CSS3 (Flexbox, Grid, медіазапити, кастомні змінні)

JavaScript (DOM, Events, LocalStorage)

Модульна структура стилів (base.css, layout.css, components.css)
